Real-time Face Recognition using CNN

Face recognition and object recognition are hot topics in computer vision and machine learning fields. This project aims to using python to implement some key algorithms about face detection and convolutional neural networks, aka CNN. The whole project can be divided into three steps.

STEP 1

Use haar or lbp cascade features in opencv to detect different faces and get enough datasets of different people. Mark different pictures with different labels. Then use FishFaceRecognizer or LBPHFaceRecognizer to learn from the datasets and train a model. Then use videos and images to check the effect of the trained model.

STEP 2

Learn basic knowledge of convolutional neural networks and tensorflow. Then, with the help of github and other resources, thry to write a CNN trainer myself and train a model. Use videos and images to chech the effect of the new model.

STEP 3

Plant what we have done to a embedded system--raspberry pi. Raspberry pi is one of the most popular demoboards, which is cheap and high-performance. Check this to get more information. This embedded system has a wide range of use, such as intelligent gate lock and monitor.
